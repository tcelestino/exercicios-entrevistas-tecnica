# Exercícios para Entrevistas Técnicas

Coleção de exercícios de algoritmos e padrões de programação organizados por linguagem. O objetivo é cobrir os problemas mais frequentes em entrevistas técnicas com implementações claras e bem documentadas.

## Linguagens

| Linguagem | Pasta | Status |
|---|---|---|
| JavaScript | [js/](js/README.md) | Disponível |
| Mais em breve | - | - |

## Como usar

Cada pasta contém um `README.md` próprio com instruções de execução, descrição dos exercícios e os conceitos envolvidos.

## Exercícios cobertos

- Estruturas de dados (HashMap, pilha, fila)
- Algoritmos clássicos (Two Sum, Fibonacci, Flatten, Palíndromo)
- Padrões assíncronos (Debounce, Throttle, Promise Pool)
- Manipulação de dados (GroupBy, DeepClone, RemoveDuplicate)
